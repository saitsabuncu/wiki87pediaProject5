# Wikipedia API Desktop Application

Python öğrenme sürecimde API kullanımı, masaüstü arayüz geliştirme ve hata yönetimi konularını uygulamalı olarak öğrenmek amacıyla geliştirdiğim bir Wikipedia arama uygulamasıdır.

Bu uygulama, kullanıcıların Türkçe olarak Wikipedia'da arama yapmasına ve ilgili konunun özet bilgilerini hızlı bir şekilde görüntülemesine olanak sağlar. Arayüz, Python'un Tkinter kütüphanesi kullanılarak geliştirilmiş olup veriler Wikipedia API üzerinden alınmaktadır.

## Features

* Search Wikipedia topics in Turkish
* Simple and user-friendly desktop interface
* Clear input and output fields
* Scrollable result area
* Error handling for invalid searches
* Fast access to Wikipedia summaries

## Technologies Used

* Python 3
* Tkinter
* Wikipedia Python Library

## Installation

Install the required library:

```bash
pip install wikipedia
```

Clone the repository:

```bash
git clone https://github.com/saitsabuncu/wiki87pediaProject5.git
cd wiki87pediaProject5
```

Run the application:

```bash
python main.py
```

* `main.py` — Application logic and graphical user interface
